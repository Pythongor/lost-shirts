.flex-container {
    display: flexbox;                   контейнер стал флексбоксом
    flex-direction:     row;            по умолчанию
                        column;
                        row-reverse;
                        column-reverse;
    justify-content:    flex-start;     элементы прижаты к началу; значение по умолчанию
                        flex-end;       элементы прижаты к концу
                        cener;          элементы выровнены по центру
                        space-between;  пространство между элементами
                        space-around;   пространство по краям равно половине отступа между элементами
                        space evenly;   пространство по краям и между элементами
    align-items:        stretch;        элементы растянуты поперёк; значение по умолчанию
                        flex-start;     элементы прижаты к началу поперечной оси
    flex-wrap:          wrap;           перенос при нехватке места
    align-content:      space-around;   пространство по краям равно половине отступа между линиями контента; значение по умолчанию
}

.item {width: auto;}

.item:nth-child(2) {flex-grow: 1;}      установлена пропорция для всего пространства, занимаемого элементами флексбокса, 
                                        имеющими это свойство. Если такой элемент один, он займёт всё возможное пространство,
                                        подвинув соседей без этого свойства максимально возможно на линии. Иначе будет 
                                        высчитываться пропорция с другими такими элементами
